基于Hugo创建博客

第一步：安装Hugo

下载地址：hugo_extended

  1. 下载适合你的操作系统的版本。
  2. 把 hugo（或者是 Windows 的 hugo.exe）放到你的环境变量 PATH 所在的目录。

第二步：创建站点及文章

Hugo创建站点

$ hugo new site 你的项目名

创建一个文章内容页面

$ hugo new index.md

第三步：安装皮肤

选择喜欢的皮肤下载，在 \themes\ 目录下

$ git clone https://github.com/olOwOlo/hugo-theme-even themes/even

下载地址：

  • Hugo皮肤库
  • 本博客皮肤-even

注意： even皮肤要客制化主题使其能够生效，需要下载 hugo_extended Hugo定制版。

例如：我想要将even原本默认的主题颜色 Default 修改为 Cobalt Blue ，只需修改

\themes\even\assets\sass 路径下的 _variables.scss 文件，然后再在网页主目录下使用hugo命令即可构建成功。

第四步：运行Hugo

Hugo 包含了内置的高性能 web server 。通过简单的运行 hugo server ， Hugo 将会找一个有效的端口运行服务器用于你的文章。

第五步：生成静态文件

使用 Hugo 命令生成静态文件

注意：需要把草稿文件draft: true 改为 draft: false ，否则静态文件是不生成当前文章。
