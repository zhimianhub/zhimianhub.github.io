Hugo常见命令

  1. 创建项目及文章：

    hugo new ...

    • hugo new site 项目名
    • hugo new ....md

  2. hugo

    hugo 本身就是一个命令，他的作用就是生成静态网站，默认在生成的静态文件保存在 public 目录中。

  3. hugo自带一个web服务器，运行 hugo server 后可以通过 http://localhost:1313 来访问静态网站。

    hugo server

    • hugo server -p 端口 : 修改默认端口
    • hugo server -D : 在使用server预览网站时，draft属性为ture的草稿文件是不会生成预览的， 添加-D后可以预览草稿文件。
