hugo-theme-even主题皮肤安装

在 \themes\ 目录下安装

$ git clone https://github.com/olOwOlo/hugo-theme-even even

重要: 在主题的 exampleSite 目录下有一个 config.toml 文件，将这个 config.toml 文件复制到你的站点目录下，根据自己的需求更改即可。

重要: 本主题用到了 Hugo Pipes 功能。如需修改 assets 目录下的文件，请安装 Hugo extended 版。

注意: 对于这个主题，你应该使用 post 而不是 posts，即 hugo new post/some-content.md。

  4. 更改even主题

    1
2

    // Default | Mint Green | Cobalt Blue | Hot Pink | Dark Violet
$theme-color-config: 'Cobalt Blue';

    将even原本默认的主题颜色 Default 修改为 Cobalt Blue ，只需修改

    \themes\even\assets\sass 路径下的 _variables.scss 文件，然后再在网页主目录下使用hugo命令即可构建成功。

    注意： even皮肤要客制化主题使其能够生效，需要下载 hugo_extended Hugo定制版，否则不会生效。
